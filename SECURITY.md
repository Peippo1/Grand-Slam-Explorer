# Security Policy

The following versions of Grand-Slam-Explorer are currently receiving security updates:

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Reporting Email:** tim.finch8891@gmail.com

### What to include in your report

- A clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes or patches

### What to expect

- You will receive an acknowledgment within 72 hours
- Verified vulnerabilities will be addressed promptly
- You will receive updates as the issue progresses
- If a report is declined, we will provide a clear explanation

Thank you for helping to keep this project secure.
